# Burst Monitor Dashboard

This repository hosts published static HTML snapshots of the Burst Monitor dashboard for public viewing.

Burst Monitor is a market-structure monitoring dashboard. The materials published here are read-only exports of the dashboard interface, updated as new snapshots are generated.

## About This Repository

The purpose of this repository is to provide access to dashboard snapshots without exposing the underlying application, codebase, or internal tooling used to generate them.

Contents may include:

- current dashboard snapshot files
- archived snapshot folders
- static assets required for rendering

Contents do not include:

- source code
- Python scripts
- backend services
- data pipelines
- proprietary research methods
- internal development assets

## Viewing

The files in this repository are intended to be viewed as static dashboard outputs.
